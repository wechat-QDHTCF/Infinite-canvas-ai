## 📖 项目简介
飙算画影网是一套面向电商商家、设计师、自媒体、装修从业者的**全场景AI视觉创作开源解决方案**，核心自研**无限电商画布**工作流，内置IMAGE2、Seedream5.0、NANO BANANA三大中文AI绘画大模型，覆盖图片生成、商品美工、人像精修、家装渲染、短视频批量生成、AI数字人等全套功能，支持本地私有化部署、二次定制开发，同时开放渠道合伙人招商体系。

平台主打低成本批量产出商用视觉素材，无需PS、剪辑软件，零基础即可完成电商主图、详情页、带货短视频、家装效果图、写真海报、动漫插画，4K高清导出JPG/PNG/PSD/MP4全格式。

### 项目描述
飙算画影网一站式AI图片视频创作平台，搭载IMAGE2、Seedream5.0、NANO BANANA绘画模型，无限电商画布批量生成商品主图、详情页、带货短视频、AI数字人、家装效果图，支持模特换背景、产品精修、老照片修复，可本地私有化部署，同步招募渠道合伙人，高算力返佣一对一运营扶持。

## ⭐ 核心亮点
1. **超大无限电商画布**
无边界可视化画布，自定义工作流，批量堆叠商品素材、分镜、海报，支持图层管理、局部重绘、素材拖拽排版，整套电商视觉一站式落地。
2. **三大国产中文AI大模型自由切换**
- IMAGE2：全能通用生图，中文理解强，全场景高清出图
- NANO BANANA(NB)：电商质感模型，复杂商品场景、高级氛围感海报首选
- Seedream5.0：电影级画质，人像、短视频分镜文字还原精准
3. **电商全链路自动化美工**
专为淘宝、拼多多、抖音商家打造，批量白底图裂变、自动详情页、促销海报、产品精修，大幅降低美工人力成本。
4. **图文+视频一体化生成**
单图转4K带货短片、爆款视频复刻、AI数字人口播、宠物对口型视频，智能运镜转场，批量产出短视频素材。
5. **多行业通用工具集**
人像写真、证件照、家装3D渲染、动漫二次元、老照片修复、AI抠图去水印，设计师、装修公司、自媒体通用。
6. **商用导出+渠道合伙人体系**
支持高清商用格式输出；开放独立品牌代理权限，算力高额返佣、全套画布运营教学、一对一陪跑扶持。

## 🚀 本地部署指引
1. 克隆仓库
```bash
git clone https://github.com/wechat-QDHTCF/Infinite-canvas-ai.git
cd Infinite-canvas-ai
```
2. 安装依赖
```bash
npm install
# 或 pnpm install
```
3. 本地启动
```bash
npm run dev
```
4. 生产打包
```bash
npm run build
```
